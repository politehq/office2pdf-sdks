# Security Policy

The Office2PDF team takes security issues seriously.  
We appreciate responsible disclosure and will work with you to address any vulnerabilities.

---

## Reporting a Vulnerability

If you believe you have found a security vulnerability in Office2PDF or any of its SDKs, please report it privately.

**Do not open a public GitHub issue.**

Instead, contact us at:

ðŸ“§ **admin@politeai.cloud**

Please include as much detail as possible:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any relevant logs, screenshots, or proof-of-concept code

---

## Response Timeline

We aim to respond to security reports within **48 hours**.

If the issue is confirmed, we will:

- Investigate and assess the impact
- Prepare and release a fix
- Coordinate disclosure when appropriate

---

## Supported Components

Security updates apply to:

- Official Office2PDF API
- Official SDKs maintained in this repository

Third-party integrations or unofficial forks are outside the scope of this policy.

---

Thank you for helping keep Office2PDF and its users safe.
